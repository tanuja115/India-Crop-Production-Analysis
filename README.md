# India Crop Production Analysis (2020-2025)

## Project Objective
An interactive Power BI dashboard developed during my internship to analyze agricultural trends across India, focusing on production volume, land area usage, and yield efficiency.

## Key Features
* *Overview Page:* Geo-spatial mapping of production by state and high-level KPIs.
* *Crop Analysis:* Breakdown of crop distribution and yearly production trends.
* *State Performance:* Ranking of states using Funnel charts and detailed data tables.

## Tools Used
* *Power BI* (Data Visualization & DAX)
* *Excel* (Data Source)
* *GitHub* (Project Documentation)

## Insights
* Identified the top-producing states using funnel visualization.
* Analyzed the production share of major crops like Rice, Wheat, and Sugarcane
